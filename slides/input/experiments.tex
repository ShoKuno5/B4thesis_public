      \begin{column}{.5\textwidth}
        \begin{block}{Rössler方程式}
            次の方程式を解くことで時系列データを生成する．
            \begin{align}
                \frac{dx}{dt} &= -y - z + P(t)\\
                \frac{dy}{dt} &= x + ay \\
                \frac{dz}{dt} &= b + z(x - c)
            \end{align}
            ここで，$P(t) := A \sin(t + \theta_p(t))$ とする．
           
            ただし，$p \in \left\{ n \in \Z \mid -12 \leq n \leq 12 \right\}$ とし，
            $\theta_p$ は $4$ 日に $1$ 度外力の位相を $p\cdot 2\pi/24$ だけ早めるような関数．
        \end{block}
      \end{column}
